# Tips Memilih Ukuran Kasur Sewaan yang Tepat

Salah ukuran kasur bikin tidur tidak nyaman. Panduan memilih ukuran yang pas untuk kebutuhan Anda.

## Ukuran Standar

- **Single (90x200)** — untuk satu orang dewasa. Cocok untuk kost, kamar sempit
- **Double (120x200)** — lebih lega untuk satu orang. Pas untuk kontrakan kecil
- **Queen (160x200)** — standar pasangan. Nyaman untuk dua orang
- **King (180x200)** — mewah dan luas. Untuk kamar besar atau keluarga dengan anak kecil

## Cara Memilih

1. Ukur luas kamar — sisakan minimal 60cm di sisi kasur untuk jalan
2. Pertimbangkan jumlah pengguna — single untuk sendiri, queen/king untuk berdua
3. Tinggi badan — tambah 20cm dari tinggi badan untuk panjang kasur
4. Akses masuk — pastikan kasur bisa masuk pintu dan tangga (terutama untuk kost lantai 2)
