# Sewa Kasur untuk Tamu Menginap Lebaran di Jogja

Lebaran tiba, sanak saudara datang dari luar kota. Kamar terbatas, kasur kurang. Solusinya: sewa kasur tambahan.

## Kenapa Sewa untuk Musim Lebaran?

- Pesan untuk 3-14 hari saja — tidak perlu beli
- Antar-jemput sebelum dan sesudah Lebaran
- Pilih jumlah sesuai kebutuhan tamu
- Kasur bersih dan siap pakai

## Tips Persiapan

1. Hitung jumlah tamu yang akan menginap
2. Pesan minimal 3 hari sebelum Lebaran (kuota terbatas)
3. Siapkan ruang alternatif: ruang tamu, ruang keluarga
4. Kombinasikan dengan paket extra bed untuk rombongan besar

Layanan ini cocok untuk tamu mudik, keluarga besar, atau acara halal bihalal yang berlangsung beberapa hari.
